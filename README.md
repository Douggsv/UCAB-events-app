# 🎓 UCAB Events App

A web application prototype designed for Universidad Católica Andrés Bello (UCAB) to centralize and showcase university events through a clean and user-friendly interface.

This project focuses on UI/UX design and frontend structure, providing a conceptual solution for event management within an academic environment.

---

## 📌 Overview

This academic project presents a web-based platform where students can easily explore upcoming university events.

The application emphasizes:

* Clear information hierarchy
* Simple and intuitive navigation
* Clean and responsive design

---

## 🚀 Features

* 📅 Event listing interface
* 🧭 Easy navigation between sections
* 🎨 Clean UI design using Bootstrap
* 📱 Responsive layout for different devices
* 🏫 University-focused event structure

---

## 🛠️ Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling
* **Bootstrap** – Responsive design framework
